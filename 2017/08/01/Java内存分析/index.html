Java内存分析

公司生产环境有内存问题，同事总结的命令

jmap -histo:live [pid]
这个命令，可以显示当前堆信息
jstat -gcutil [pid]
这个命令，可以打印内存gc情况
jmap -dump:live,format=b,file=[path] [pid]

这个命令，是生成heap dump文件，后续可以分析工具分析

dump分析工具

  • eclipse MAT
  • jvisualvm
